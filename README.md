# Task 2 - Network Security & Scanning

This project demonstrates basic network reconnaissance, port scanning, vulnerability assessment, and traffic analysis in a controlled lab environment.

## 🎯 Objective

To identify open ports, detect running services, analyze network traffic, and understand common vulnerabilities using industry-standard tools.

## 🛠 Tools Used

* Nmap
* Wireshark
* Nessus / OpenVAS
* iptables

## 🖥 Target System

* **Metasploitable2**
* IP Address: 192.168.64.3
* Environment: Local Virtual Lab (Kali Linux)

## 🔍 Activities Performed

* Host discovery using ping
* Port and service scanning using Nmap
* Basic vulnerability assessment
* Packet capture and analysis using Wireshark
* Firewall rule implementation using iptables
